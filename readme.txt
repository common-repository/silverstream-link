This plugin adds a button to TinyMCE to allow restaurants powered by Wordpress to easily add a link to the silverstream.io booking system.

== Description ==

Once installed, you may edit any page or post and click the "Silverstream Link" button to insert a link to Silverstream. You will be prompted to enter your Restaurant Id (e.g. ABC123).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/silverstream-link` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
